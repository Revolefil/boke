## 简介
> 自从使用了 MVVM 框架来写 Web 页面后，直接操作 DOM 的机会非常少了。因为视图和模型的绑定，我们对数据的操作会自动更新到页面，使我们从繁琐的 DOM 操作中解放了出来，因此我们可以通过复杂的数据操作，来实现以前很难处理的页面。这篇文章会通过 Vue 来学习 JavaScript 对基本数据类型的操作，并实现一些有意思的小 Demo。

## 教程
> 网上见到太多的讲 JavaScript 基础数据类型操作的博客，大部分都是每个函数每个方法的列出来，讲下作用，讲下参数，然后配个运行结果的小例子就完事了。这样的教程会发现看过去后映像并不会太深，因为很难与实际会遇到的问题联系起来，或者不知道这个函数可以解决工作中的哪些问题。鉴于此，我于是决定用需求驱动的方式来学习 JavaScript 的数据结构操作。

本文档并不会涉及太复杂的数据结构（太复杂的我也不会），只涉及到 JavaScript 的一些基本数据类型和一些 ES6 提供的新方法。
